# user-interaction-system
## 1.项目背景

用golang实现一个简单用户互动系统





## 2.基础功能模块

1. 发布评论：支持无限盖楼回复。

2. 读取评论：按照时间、热度排序；显示评论数、楼中楼等。

3. 删除评论：用户删除、UP主删除等。

4. 评论互动：点赞、点踩、举报等。

5. 管理评论：置顶、精选、后台运营管理（搜索、删除、审核等）。

评论一般还包括一些更高阶的基础功能：

1. 评论富文本展示：例如表情、@、分享链接、图片等。

2. 评论标签：例如UP主点赞、UP主回复、好友点赞等。

3. 评论装扮：一般用于凸显发评人的身份等。

4. 热评管理：结合AI和人工，为用户营造更好的评论区氛围。
